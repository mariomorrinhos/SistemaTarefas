# SistemaTarefas
Sistema de Gestão de Tarefas

Para o uso do sistema, abra o arquivo config/database/conexao.php e coloque o nome do banco de dados, o usuário e a senha. Em seguida, aba o arquivo configurar_banco_de_dados.php na pasta raiz diretamente no navegador. Ele criará as tabelas necessárias.
